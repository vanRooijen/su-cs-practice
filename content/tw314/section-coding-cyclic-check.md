<div id="content" class="pretext-content"><section class="section" id="section-coding-cyclic-check"><h2 class="heading hide-type">
<span class="type">Section</span> <span class="codenumber">11.3</span> <span class="title">The check polynomial and the parity-check matrix of a cyclic code</span>
</h2>
<p id="p-644">Let \(C\) be a cyclic \([n,k]\)-code with generator polynomial \(g(x)\text{.}\) Since \(g(x)\) is a factor of \(x^{n}-1\text{,}\) \(x^{n}-1=g(x)h(x)\) for some polynomial \(h(x)\text{.}\)</p>
<p id="p-645">Since \(g(x)\) is monic, so is \(h(x)\text{.}\) Furthermore \(g(x)\) has degree \(n-k\text{,}\) so \(h(x)\) has degree \(k\text{.}\) The polynomial \(h(x)\) is called the <dfn class="terminology">check polynomial</dfn> of \(C\text{.}\)</p>
<article class="theorem theorem-like" id="theorem-coding-check"><h6 class="heading">
<span class="type">Theorem</span><span class="space"> </span><span class="codenumber">11.3.1</span><span class="period">.</span>
</h6>Let \(C\) be a cyclic code in \(\mathbf{R}_n\) with generator polynomial \(g(x)\) and check polynomial \(h(x)\text{.}\) Then \(c(x)\in C\) if and only if \(c(x)h(x)=0\text{.}\)</article><p id="p-646">Note that \(h(x)\) does NOT always generate the dual code \(C^\perp\text{.}\)</p>
<article class="theorem theorem-like" id="theorem-coding-paritycheck"><h6 class="heading">
<span class="type">Theorem</span><span class="space"> </span><span class="codenumber">11.3.2</span><span class="period">.</span>
</h6>Let \(C\) be a cyclic \([n,k]\)-code with check polynomial<div class="displaymath">
\begin{equation*}
h(x)=h_0+h_1x+\cdots+h_kx^k.
\end{equation*}
</div>Then a parity-check matrix for \(C\) is<div class="displaymath">
\begin{equation*}
H=\begin{bmatrix}
h_k&amp;h_{k-1}&amp;\cdots&amp;h_0&amp;0&amp;0&amp;\cdots&amp;0\\
0&amp;h_k&amp;h_{k-1}&amp;\cdots&amp;h_0&amp;0&amp;\cdots&amp;0\\
0&amp;0&amp;h_k&amp;h_{k-1}&amp;\cdots&amp;h_0&amp;\cdots&amp;0\\
\vdots&amp;\vdots&amp;&amp;\ddots&amp;\ddots&amp;&amp;\ddots&amp;\vdots\\
0&amp;0&amp;\cdots&amp;0&amp;h_k&amp;h_{k-1}&amp;\cdots&amp;h_0
\end{bmatrix}
\end{equation*}
</div>and \(C^\perp\) is a cyclic code generated by the polynomial<div class="displaymath">
\begin{equation*}
h'(x)=h_k+h_{k-1}x+\cdots+h_0x^k.
\end{equation*}
</div></article><article class="example example-like" id="example-coding-cyclicpoly"><h6 class="heading">
<span class="type">Example</span><span class="space"> </span><span class="codenumber">11.3.3</span><span class="period">.</span><span class="space"> </span><span class="title">Generator matrix of a cyclic code.</span>
</h6>
<ol class="decimal">
<li id="li-221">Find the generator polynomial of the dual of the ternary \([9,7]\)-code \(C\) that is generated by \(g(x)=x^2+x+1\text{.}\)</li>
<li id="li-222">Find the generator polynomial of the dual of a 7-ary \([3,1]\)-code \(C\) that is generated by \(g(x)=x^2+4x-5\text{.}\)</li>
</ol>
<div class="solutions">
<span class="id-ref original"><span class="type">Solution</span></span><div class="hidden-content tex2jax_ignore" id="hk-solution-60"><div class="solution solution-like"><ol class="decimal">
<li id="li-223">Use long division to find that \(x^9-1=(x^2+x+1)(x^7-x^6+x^4-x^3+x-1)=g(x)h(x)\text{.}\) From <span class="xref">Theorem 9.6.6</span> \(C^\perp\) is generated by \(h'(x)=-x^7+x^6-x^4+x^3-x+1\text{.}\) Since the generator polynomial of a code must be monic, the generator polynomial of \(C^\perp\) is \(g'(x)=-h'(x)=h(x)\text{.}\)</li>
<li id="li-224">Since \(x^3-1=(x^2+4x-5)(x-4)=g(x)h(x)\text{,}\) it follows from <span class="xref">Theorem 9.6.6</span> that \(C^\perp\) is generated by \(h'(x)=-4x+1=3x+1\text{.}\) Since the generator polynomial of a code must be monic, the generator polynomial of \(C^\perp\) is \(g'(x)=5(3x+1)=x+5\neq h(x)\text{.}\)</li>
</ol></div></div>
