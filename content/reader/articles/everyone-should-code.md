## Everyone Should Code

The department feature page highlights code.org's message that every student should have access to computer science education.

The source notes that code.org is a non-profit focused on expanding participation in Computer Science, including increased participation by women and underrepresented groups.

It argues that Computer Science should be treated alongside core STEM subjects in school curricula.
