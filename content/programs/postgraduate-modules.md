## Postgraduate Modules

The postgraduate modules page outlines:

- Honours in Computer Science (coursework + project)
- Masters in Computer Science (thesis-based)
- PhD in Computer Science (dissertation-based)

### Honours structure

The source describes six elective 16-credit modules plus a 32-credit honours project, with module offerings varying by year.

### Module examples listed on the page

- Advanced Algorithms
- Artificial Intelligence
- Machine Learning variants
- Computer Networks
- Functional Programming
- Software Testing and Analysis
- Automata Theory and Applications
